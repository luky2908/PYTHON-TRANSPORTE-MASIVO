Introducción
En este programa, se implementa un sistema de transporte masivo utilizando un grafo para representar estaciones y sus conexiones. Cada conexión tiene un tiempo de viaje y un costo asociado. Para encontrar la mejor ruta entre dos estaciones, se utiliza una variación del algoritmo de Dijkstra, que emplea una cola de prioridad para siempre procesar primero la ruta más corta o económica.

El sistema permite:

Agregar estaciones y conexiones con tiempo y costo.
Buscar la mejor ruta entre dos estaciones, minimizando el tiempo y/o el costo.
Imprimir la ruta óptima, junto con el tiempo estimado y el costo total del viaje.
Este enfoque es útil para modelar redes de transporte, como sistemas de metro, autobuses o trenes, y optimizar los desplazamientos de los usuarios.
